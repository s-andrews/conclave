Figures and Graphs

In addition to the main areas of the SeqMonk display there are a number of additional figures and graphs which can be produced on demand. These can help to illustrate different properties of your data.

The currently available additional figures are:

Histograms and distribution summaries

  • The Probe Value Histogram
  • The Probe Length Histogram
  • The Reading Length Histogram
  • The BoxWhisker Plot
  • The Star Wars Plot
  • The Cumulative Distribution Plot

Clustering and correlation

  • The DataStore Tree
  • The Correlation Matrix
  • The Probe List Overlap Matrix
  • The Venn Diagram Plot
  • The Hierarchical Cluster Plot

Quantitation aggregation and summaries

  • The Probe Trend Plot
  • The Aligned Probes Plot
  • The Quantitation Trend Plot
  • The Domainogram Plot

Comparison plots

  • The Scatter Plot
  • The MA Plot
  • The Line Graph

Hi-C Specific Plots

  • The HiC Cis-Trans Scatterplot
  • The HiC Length Histogram
  • The HiC Heatmap Plot

QC Plots

  • The RNA-Seq QC plot
  • The Small RNA QC plot
