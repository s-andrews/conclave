Intersect Existing Lists

The intersect list tool allows you to logically combine multiple probe lists, to specify inclusion or exclusion criteria from the final list. If you just want to combine two lists with more complex logic, use the "Logically combine existing lists".

In this filter you simply say for each probe list whether a probe must be present within the list, excluded from the list, or if that list should not be considered. Simply tick the include or exclude boxes for the lists you care about.
